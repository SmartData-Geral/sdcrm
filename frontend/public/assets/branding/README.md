# Branding do sistema

Coloque aqui o favicon oficial do projeto.

Arquivos esperados:

- `favicon.svg` (preferencial)
- `favicon.ico` (opcional para compatibilidade)

O `index.html` da aplicacao esta configurado para usar:

- `/assets/branding/favicon.svg`

Se optar por usar `favicon.ico`, atualize o `href` no `index.html`.
